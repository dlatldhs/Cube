# Cube 
> ### 파이썬을 사용하여 만든 간단한 CD 큐브이다.
## 개발 동기
> ### 이 프로젝트를 진행할 당시 게임 쪽에 흥미가 있었고 일단 간단한 파이썬 라이브러리들을 다루며 스킬을 쌓고 싶었기에 3D큐브를 구현을 해보기로 했다.
## 기대효과
> ### 강의 영상 및 블로그 등을 보면서 이러한 기능들은 어느 형식으로 해결하는지 해결능력을 키울 수 있었다.
